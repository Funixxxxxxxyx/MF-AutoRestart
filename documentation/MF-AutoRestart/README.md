# Introduction

Welcome to the official documentation for **MF-AutoRestart**.

MF-AutoRestart is a premium, lightweight, and highly configurable Paper 1.21.x plugin designed to perform safe and automated restarts on Minecraft servers and networks.

---

## Why MF-AutoRestart?

Many auto-restart scripts abruptly kick players, disconnect active processes, and can cause player data loss or database corruptions. MF-AutoRestart prevents this by coordinating shutdown states gracefully.

### Key Objectives
* **Safe shutdowns**: Execute warning intervals, dispatch console saves, and redirect players smoothly.
* **Network integration**: Send players to lobby servers instead of kicking them during a restart window.
* **Admin control**: Add, check, and delete schedules dynamically in-game without changing configs.
* **Extensible architecture**: Integrate with PlaceholderAPI (PAPI) for custom HUDs and hook external plugins into custom Bukkit lifecycle events.
